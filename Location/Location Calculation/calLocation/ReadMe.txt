1、The json_to_csv file converts JSON files generated by the network into CSV files that can be read and processed.
2、The calLocation.py file contains code for batch computing coordinates; simply modify the paths of the three folders to use it.
3、The mergecsv file merges the computed CSV files into one file.
4、The selectJson file selects the corresponding JSON files that need location calculation and copies them to the LNbbox folder.
5、The biomassXY file extracts the X and Y data from the output_csv file and appends it to the structural calculation results file.
6、The project file converts WGS84 coordinates to UTM for calculating the distance between points.
7、The cluster file performs clustering on the original computed coordinate points.
